PreRequisites:
  Expects Data file and Code to be on the same path.

Description:
Code has the following functions:
  1. fieldSortAndTop(data, field="Current Rent", top=5): produce a list sorted by Current Rent in ascending order .
  2. filterOnLeaseYears(data, i=25): Creates a new list of mast data with Lease Years = 25 years.
  3. groupByField(data, comment=""): Create a dictionary containing tenant name and a count of masts for each tenant.
  4. groupByFieldCaseInsenstve(data): same as above but case insensitive
  5. filterOnLeaseSD(data): Lists the data for rentals with Lease Start Date between 1 June 1999 and 31 Aug 2007.
  6. all(data):	calls the functions fieldSortAndTop(data),filterOnLeaseYears(data),groupByField(data) and filterOnLeaseSD(data)
  7. choice(): prints the choice text
  8. Main:Takes the choice which includes performing all the functions or none and exit.
